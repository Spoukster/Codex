# Визуальный стиль артефактов (единый)

Да, теперь визуал задаётся **в одном стиле** для всех артефактов.

## Стиль-библия
- Стиль: **neo-retro flat illustration**.
- Контур: толстый чистый outline.
- Свет: мягкие градиенты без фотореализма.
- Цвет: приглушённая палитра + акценты по эпохам.
- Фон: минимальный, слегка текстурный (grain/paper).
- Композиция: один предмет по центру, без перегруза деталями.

## Цветовые акценты по эпохам
- Прошлое: тёплые земляные тона (охра, бронза, сепия).
- Настоящее: нейтрально-городские (графит, синий, мята).
- Будущее: холодные неон-акценты (cyan/violet/magenta).

## Промпты для генерации
Промпт каждого артефакта хранится в `Artifact.imagePrompt` в `GameViewModel` и уже выровнен под единый стиль.

Файл-источник промптов:
- `app/src/main/java/com/codex/chronosorter/GameViewModel.kt`
