# Chrono Sorter — оригинальная идея мобильной игры-сортера

## Коротко
Игрок — стажёр «Хронопочты», который сортирует артефакты по временным порталам: **прошлое**, **настоящее**, **будущее**. Каждый неверный выбор ломает временную линию.

## Почему это не просто обычный сортер
- У каждого предмета есть «подсказка из лора», а не только очевидная картинка.
- Комбо-механика: серия правильных сортировок даёт бонус к очкам.
- Можно расширить до событий: «временные аномалии», когда привычные предметы временно смещаются в соседнюю эру.

## Игровой цикл
1. Появляется артефакт + короткая подсказка.
2. Игрок выбирает один из 3 порталов.
3. Получает мгновенный фидбек.
4. Игра ускоряется по мере роста счёта.

## MVP (реализовано в коде)
- 3 корзины/эпохи: прошлое, настоящее, будущее.
- Набор артефактов с подсказками.
- Очки, жизни, комбо, экран поражения и рестарт.
