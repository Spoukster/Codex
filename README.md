# Chrono Sorter (Android)

Оригинальная игра-сортер для Android: сортируй артефакты по эпохам и стабилизируй временную линию.

## Идея
Игрок работает на «Хронопочте» и отправляет предметы в нужную эпоху:
- 🏺 Прошлое
- 📱 Настоящее
- 🛸 Будущее

Ошибки уменьшают жизни. Правильные ответы дают очки и комбо-бонус.

Подробности концепта: `docs/chrono-sorter-concept.md`.

Визуальный стиль артефактов: `docs/artifact-visual-style.md`.

## Стек
- Kotlin
- Jetpack Compose
- Material 3

## Запуск
1. Открыть проект в Android Studio (Giraffe+ / Koala+).
2. Дождаться Gradle Sync.
3. Запустить `app` на эмуляторе или устройстве.
